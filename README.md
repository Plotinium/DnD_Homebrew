# DnD_Homebrew

Homebrew creations compatible with 5etools JSONs.

## Import Repository

To import my repository you just have to paste the following link after clicking `Load from URL` here [5etools Homebrew Manager](https://5e.tools/managebrew.html)

``` Text
https://raw.githubusercontent.com/Plotinium/DnD_Homebrew/main/dist/homebrew-bundle.json
```

## Useful Links

- If you need to know how to build custom json with 5etools schema, [this wiki](https://wiki.tercept.net/en/home) can solve all your problems.
- If you need some examples for your home made content, this [homebrew repository](https://github.com/TheGiddyLimit/homebrew) is the official one.
